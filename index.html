About

I've taken coding seriously since undertaking my Computer Science degree in 2018. During the degree I learned mostly Java and worked with SQL on various projects either alone or with a scrum team. However, the first time I actually touched code was working with Javascript as a teenager to make mods for minecraft as a hobby.

I've always had an aptitude and interest in coding since a young age, which I believe to stem from my interest in gaming. As I spent a lot of my free time gaming, it was natural for me to want to learn how to make games. Consequently, I undertook a Games Design Masters degree in 2022. The language I focused on the most during this time was C# in Unity Engine.

Since finishing my education, I have been working on other areas of programming. I have been coding websites and various projects using Javascript, HTML & CSS, some of which are displayed on this portfolio. Enjoy!
